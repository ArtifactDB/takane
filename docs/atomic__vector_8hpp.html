enum class takane::atomic_vector::Type
strong

Type of the atomic vector.

  • INTEGER: a number that can be represented by a 32-bit signed integer.
  • NUMBER: a number that can be represented by a double-precision float.
  • STRING: a string.
  • BOOLEAN: a boolean.
